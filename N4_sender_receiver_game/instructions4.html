Resumen: Cada ronda del Juego se desarrolla de la siguiente manera:

  1. El computador genera un número secreto.
  2. Jugador A envía un mensaje a Jugador B.
  3. Jugador B recibe el mensaje y adivina el número secreto.
  4. Ambos jugadores reciben retroalimentación.

Puntos importantes a recordar

  • El bono que puede ganar el Jugador A siempre es el mismo, pero tiene más probabilidad de ganarlo si el Jugador B adivina un número más alto.
  • El bono que puede ganar el Jugador B también es fijo, pero su probabilidad de ganarlo aumenta si su adivinanza está más cerca del número secreto.
  • El Jugador A no conoce el número secreto al enviar un mensaje.
  • El Jugador A puede elegir enviar cualquier número 1, 2, 3, 4, 5, 6, 7{{ if player.participant.treatment != "Babbling" }} o puede delegar la decisión al computador, quien envía el verdadero número secreto{{ endif }}.
    • {{ if player.participant.treatment != "Babbling" }}
  • El Jugador B nunca sabrá si el Jugador A delegó la decisión al computador o no.
    • {{ endif }} {{ if player.participant.treatment == "Decode" }}
  • Algunos Jugadores B necesitarán decodificar el mensaje del Jugador A antes de hacer la adivinanza.
    • {{ endif }}
