Decodificando el Mensaje

En el experimento, habrá dos tipos de Jugador B, a los que llamaremos Decodificador y Directo. El Decodificador tendrá que decodificar el mensaje enviado por el Jugador A. El Directo recibirá el mensaje directamente.

Ambos tipos recibirán el siguiente tipo de mensaje del Jugador A:

Mensaje del Jugador A:

Para recibir la mayor ganancia, deberías elegir X.

El tipo Directo observará directamente el número que el Jugador A pretendía enviar. El Decodificador verá el mensaje en forma de una expresión matemática. Las tareas de decodificación serán de dificultad similar a las que resolviste al inicio del experimento. Por ejemplo, si el Jugador A pretendía enviar el número 3, el Decodificador podría ver el mensaje Para recibir la mayor ganancia, deberías elegir ((14 − 2) × 2)/ 8).

El Decodificador debe responder dos preguntas antes de que se acabe el tiempo:

  1. ¿Cuál es el número que el Jugador A pretendía enviar?
  2. ¿Cuál es tu suposición del número secreto?
Es importante destacar que el Decodificador no puede adivinar el número secreto hasta que identifique correctamente el número que el Jugador A intentó enviar.

Importante:

  • El Jugador B tiene {{ Constants.TIME_PER_ROUND }} segundos para hacer una suposición. Si el Decodificador se queda sin tiempo, el número que el Jugador A envió se envía automáticamente como la suposición.
  • Los Decodificadores deben esperar {{ Constants.ATTEMPT_DELAY }} segundos entre intentos de decodificar la expresión matemática. No se recomienda adivinar al azar.
